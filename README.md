# TradeAcademy Landing Page Demo

A single-page, responsive, high-performance, and visually premium landing page for a trading education platform named **TradeAcademy**. 

## Features
- **Modern Fintech Design**: Flat, sleek, high-contrast typography and harmonized color palette.
- **Interactive Candlestick SVG Chart**: Custom-designed, realistic mock trading terminal with Candlesticks (bullish/bearish), Moving Averages, Volume Bars, Price/Time axes, and active hover tooltips.
- **Scroll-Reveal Animations**: IntersectionObserver fade-in & slide-up entry animations.
- **Interactive Course Tabs**: Switchable panels for Individual Courses and Combo Packs using pure CSS/JS.
- **Dynamic Count-Up Stat Counters**: Animates community members, satisfaction, and availability from 0 on viewport entry.
- **Interactive WhatsApp Support Widget**: Custom mock chat dialog that opens on tap with automated current timestamping.
- **Responsive Layout**: Designed mobile-first, including a mobile sticky bottom CTA bar and hamburger menu.

## How to Run
Simply open [index.html](index.html) in any modern web browser.
